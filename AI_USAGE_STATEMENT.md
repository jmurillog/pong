# AI Usage Statement

This project uses AI tooling for code generation assistance, drafting documentation, and implementation acceleration.

The following principles were applied:

- Human supervision over all technical decisions, code structure, and evaluation design.
- Reproducibility safeguards (fixed seeds, logged configs, runtime metadata, raw outputs).
- Transparent documentation of assumptions, metrics, and analysis methods.
- No fabricated experimental results: all reported outcomes must come from actual executed runs.

Before submission/publication, the team should:

1. Execute the full experiment matrix.
2. Reproduce analysis artifacts from raw logs.
3. Manually verify the final report tables/figures against generated files.
4. Keep this statement with the final appendix.
